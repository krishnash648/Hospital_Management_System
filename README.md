# 🏥 Hospital Management System

A modern and responsive Hospital Management System built using the MERN stack. The platform streamlines hospital operations by enabling patients to book appointments online while providing administrators with tools to manage doctors, appointments, and hospital information efficiently.

---

## 🚀 Live Features

### 👨‍⚕️ Patient Features

* Modern healthcare landing page
* Online appointment booking
* Department-wise doctor selection
* Responsive design for all devices
* About Us and hospital information pages
* Secure authentication system
* Real-time appointment request submission

### 🏥 Admin Features

* Admin authentication and authorization
* Manage patient appointments
* Doctor management system
* Appointment status tracking
* Dashboard analytics
* Secure backend APIs

### 🎨 UI/UX Features

* Premium healthcare-inspired design
* Fully responsive layout
* Modern appointment booking section

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Hospital_Management_System.git
cd Hospital_Management_System
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=4000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET_KEY=your_secret_key

JWT_EXPIRES=7d

COOKIE_EXPIRE=7
```

Start backend server:

```bash
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
